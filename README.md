🐍 Snake Game

Um projeto simples e divertido inspirado no clássico jogo Snake! 🕹️
Esta versão foi criada usando HTML, CSS e JavaScript, e pode ser jogada diretamente no navegador.

ㅤ

🎮 Sobre

No Snake Game, você controla uma cobra dentro de um ambiente limitado. O objetivo é coletar os alimentos que aparecem aleatoriamente na tela, fazendo com que a cobra cresça a cada item consumido. O jogo termina se a cobra colidir com as paredes ou com seu próprio corpo — tente bater o seu recorde!

ㅤ

📦 Estrutura do Projeto

📁 Arquivos principais

index.html - Estrutura da página e canvas do jogo

style.css - Estilo visual e layout

script.js - Lógica do jogo (movimento, colisões, etc.)

ㅤ

🚀 Como Jogar

Abra https://snake-game-beta-azure.vercel.app no seu navegador preferido.

Controle a cobra usando as setas do teclado:

⬆️ Cima

⬇️ Baixo

⬅️ Esquerda

➡️ Direita

Coma os alimentos para crescer e aumente sua pontuação.

Evite bater nas paredes ou em você mesmo — isso encerra o jogo!
